# E-Sports League - Landing Page

** Live:** https://J-Nguyen-web.github.io/e-league

Modern Single-page landing website for an E-Sports community - build only with **HTML & CSS**.
This project demonstrates layout architecture, typography, brand color mapping and component thinking in plain CSS without any JS or framework

## CSS Highlights

| Concept | Where |
|---|---|
| flexbox layout system | navigation, about columns, footer columns, schedule list|
| absolute positioning | welcome screen imagae background paralax |
| responsive typography scale | url imported fonts |
| semantic color variables (manual palette) | gold, dark bg, gray, brown accent |
| border styling as branding | left border highlight on match cards |
|custom spacing rhythm (rem) | consistent certical rhythm acros selections |

The main goal was - **component-level design without Tailwind / Bootstrap**
